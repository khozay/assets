# typeflow-assets

Downloadable content for [Typeflow](https://github.com/khozay/Typeflow) that isn't bundled into the app itself, distributed as versioned [Releases](https://github.com/khozay/assets/releases) rather than committed into this repo's tree.

Currently hosts:
- **Suggestion models** — n-gram language models used for autocomplete/suggestions (e.g. `en.bin`).

This repo intentionally stays public and empty of large binaries in its tracked history — actual files live as release assets so the app can fetch them over a plain HTTPS URL.
