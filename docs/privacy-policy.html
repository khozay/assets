Typeflow Keyboard — Privacy Policy

Last updated: 2026-07-09 (v2)

Typeflow is a virtual keyboard (Input Method Editor) for Android. This page explains what data Typeflow accesses, where it goes, and what control you have over it.

What Typeflow can see

Like any Android keyboard, Typeflow receives the text you type so it can display it, offer word suggestions, and provide autocorrect. This is a platform requirement of every keyboard app, not something unique to Typeflow.

  • Password and PIN fields are excluded. Typeflow detects password-type input fields and automatically disables suggestions, learning, and the AI assistant while one is focused.
  • Typed text used for suggestions is processed entirely on-device and is never sent anywhere.

Data stored on your device

All of the following stays in the app's private storage folder on your device and is deleted automatically if you uninstall Typeflow:

  • Keyboard settings, themes, and layout preferences
  • Clipboard history (when enabled) — capped in size, and never transmitted off-device
  • AI assistant settings (e.g. the address of your own Ollama server)

Optional local AI assistant (off by default)

Typeflow includes an optional AI assistant that connects to an Ollama server you run yourself, on your own device or local network. It is not required to use the keyboard, is off by default, and is disabled automatically while a password or PIN field is focused. Nothing you send to it leaves your own local network — Typeflow has no cloud AI provider and does not relay this text anywhere else.

Permissions Typeflow requests

  • Internet — used to (a) talk to your own local Ollama server if you enable the AI assistant, and (b) optionally download extra content you choose to install (additional keyboard layouts, emoji packs, or suggestion models), fetched directly from Typeflow's own hosting on GitHub. No typed text is ever uploaded as part of this.
  • Display over other apps — required only if you enable the floating/detached keyboard mode.
  • Vibrate — used for optional haptic feedback on key press.

Typeflow does not request access to contacts, storage, location, camera, or microphone.

What Typeflow does not do

  • No analytics, tracking, or advertising SDKs are included.
  • No account or sign-in is required to use the keyboard.
  • No data is sold, shared, or sent to any third party. Typeflow has no cloud AI backend of its own.

Your control over data

You can clear stored settings and clipboard history from the device's App Info screen (Storage > Clear data), or remove all app data by uninstalling Typeflow.
