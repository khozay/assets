Typeflow Keyboard — Privacy Policy

Last updated: 2026-07-09

Typeflow is a virtual keyboard (Input Method Editor) for Android. This page explains what data Typeflow accesses, where it goes, and what control you have over it.

What Typeflow can see

Like any Android keyboard, Typeflow receives the text you type so it can display it, offer word suggestions, and provide autocorrect. This is a platform requirement of every keyboard app, not something unique to Typeflow.

  • Password and PIN fields are excluded. Typeflow detects password-type input fields and automatically disables suggestions, learning, and AI features while one is focused.
  • Typed text used for suggestions is processed on-device and is not sent anywhere, unless you explicitly enable a cloud AI feature described below.

Data stored on your device

All of the following stays in the app's private storage folder on your device and is deleted automatically if you uninstall Typeflow:

  • Keyboard settings, themes, and layout preferences
  • Clipboard history (when enabled) — capped in size, and never transmitted off-device
  • Optional AI provider settings, including any API key you choose to enter (see below)

Optional cloud AI features (off by default)

Typeflow includes an optional "Smart Things" AI assistant with two modes, neither of which is required to use the keyboard:

  • Local (Ollama): connects to an Ollama server you run yourself on your own device or local network. Nothing leaves your local network.
  • Cloud (Hugging Face): if you supply your own Hugging Face API key and enable this mode, the text you submit to the assistant is sent to Hugging Face's inference API (router.huggingface.co) to generate a response. This only happens when you actively use the assistant — ordinary typing and suggestions are never sent here.

Both AI modes are disabled while a password or PIN field is focused.

Permissions Typeflow requests

  • Internet — required only for the optional cloud AI feature above.
  • Display over other apps — required only if you enable the floating/detached keyboard mode.
  • Vibrate — used for optional haptic feedback on key press.

Typeflow does not request access to contacts, storage, location, camera, or microphone.

What Typeflow does not do

  • No analytics, tracking, or advertising SDKs are included.
  • No account or sign-in is required to use the keyboard.
  • No data is sold or shared with third parties.

Your control over data

You can clear stored settings and clipboard history from the device's App Info screen (Storage > Clear data), or remove all app data by uninstalling Typeflow. If you entered a Hugging Face API key, clearing AI settings removes it from local storage.
